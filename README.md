# vue-dynamic-layout-demo

Dynamic different layout changes with easily way

## Getting Started

### Prerequisites

```
npm install

or

yarn
```

## Run Application

```
npm run serve

or

yarn run serve
```

## Built With

* [Vue](https://vuejs.org/) - The web framework used
* [Vue Cli](https://cli.vuejs.org/) - Project template
